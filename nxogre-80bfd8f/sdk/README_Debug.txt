Instructions for including and linking 
-------------------------------------- 
These are the following parameters you should append to your project linker settings.  
  
* Additional Include Directories (.h) C:\Desarrollo\nxogre-80bfd8f\sdk  
* Additional Library Directories (.lib) C:\Desarrollo\nxogre-80bfd8f\sdk  
* Additional Dependencies - NxOgre_Debug.lib 
 
Then copy the file C:\Desarrollo\nxogre-80bfd8f\sdk\NxOgre_Debug.dll to your application directory.  
 
For a first time user, you should follow one of the many tutorials at http://nxogre.org or have a look at Cake to set up your application.
